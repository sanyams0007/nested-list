# Nested List

A React app that generates dynamic dropdown nested list items according to input size in UI

## Run Locally

Clone the project

```bash
  git clone https://github.com/sanyams0007/nested-list.git
```

Go to the project directory

```bash
  cd nested-list
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
